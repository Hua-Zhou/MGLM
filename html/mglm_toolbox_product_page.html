MGLM Toolbox

The MATLAB MGLM Toolbox is a collection of Matlab functions for multi-response GLM regression and sparse regression.

Documentation

  • Dirichlet-Multinomial distribution
  • Generalized Dirichlet-Multinomial distribution
  • Negative multinomial distribution
  • Multinomial-logit regression and sparse regression
  • Dirichlet-Multinomial regression and sparse regression
  • Generalized Dirichlet-Multinomial regression and sparse regression
  • Negative multinomial regression and sparse regression
